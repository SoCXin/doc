STM32 Cortex-M4
=====================

Helper command for :ref:`mcs51`.

To print all available commands and options use:

.. code-block:: bash

    pio remote --help
    pio remote COMMAND --help

    # run command on the specified PlatformIO Remote Agents
    pio remote --agent NAME_1 --agent NAME_N COMMAND
